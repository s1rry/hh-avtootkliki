# hhautoapply — авто-отклики на hh.ru в Telegram

Бот сам ищет вакансии по твоим фильтрам и откликается за тебя 24/7: подключаешь
hh.ru по коду (без пароля), заводишь задачу — бот шлёт отклики, персональные
сопроводительные письма (ИИ), поднимает резюме и показывает воронку
(отклики → просмотры → приглашения).

## Возможности

- **Задачи** — у каждой свой ключ поиска, своё резюме и свои настройки
  (регион, формат, опыт, занятость, зарплата, лимит, часы, стоп-слова).
- **Источник вакансий** — поиск по ключу и/или **лента рекомендаций hh** под
  резюме (`/similar_vacancies`): её в разы больше, и она сама обновляется.
- **Умный отбор (ИИ)** — оценивает вакансию по резюме и отсекает нерелевантные.
- **Сопроводительные письма** — ИИ пишет под каждую вакансию, или своё готовое
  письмо. Есть **A/B-тест** писем по приглашениям.
- **Воронка по задаче** — отправлено / просмотры / приглашения (из hh
  `/negotiations`), плюс вечерний дайджест за день.
- **Авто-поднятие резюме**, мультиаккаунт, статистика по задаче и общая.
- **Вход по коду (OTP)** — телефон → код от hh. Пароль не нужен и не хранится.

## 🚀 Self-host (Linux / VPS, один клик)

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/s1rry/hhautoapply/install-docs/scripts/install.sh)
```

Скрипт поставит Python 3.12, зависимости, Playwright и создаст `.env`.
Дальше впиши три ключа и запусти. Подробно (в т.ч. macOS/Windows) — [INSTALL.md](INSTALL.md).

### Минимальный `.env`

```
MODE=single                      # одиночный режим (всё бесплатно, для себя)
TG_BOT_TOKEN=...                 # токен своего бота от @BotFather
TG_ADMIN_CHAT_ID=...             # твой Telegram ID

# ИИ — любой OpenAI-совместимый провайдер (по умолчанию Cerebras, бесплатно)
AI_PROVIDER=openai
AI_API_KEY=...
AI_BASE_URL=https://api.cerebras.ai/v1
AI_MODEL=gpt-oss-120b
```

hh подключается прямо в боте: `/connect` → телефон → код. Пароль не нужен,
работает headless (без VNC).

Два режима из одной кодовой базы:
- **`MODE=single`** — self-host для себя, весь функционал бесплатно.
- **`MODE=multi`** — мультиюзер (тарифы, оплата) — так работает облачный бот.

## Как пользоваться

1. `/start` → **🔗 Подключить hh.ru** (телефон → код).
2. **📋 Задачи → ➕ Новая задача**: выбери резюме и ключ поиска.
3. В карточке задачи — источник (ключ / рекомендации), фильтры, письма, лимит.
4. Бот сам откликается по расписанию. Смотри воронку на карточке и в дайджесте.
